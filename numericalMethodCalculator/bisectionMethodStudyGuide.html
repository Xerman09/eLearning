Bisection Method Study Guide

1. Understanding the Bisection Method

The bisection method is one of the simplest and most reliable root-finding algorithms. It is based on repeatedly dividing an interval in half and selecting the subinterval where the root lies.

For a continuous function \(f(x)\) on the interval \([a, b]\) where \(f(a) \cdot f(b) < 0\) (opposite signs):

\(c = \frac{a + b}{2}\) (midpoint)

The method works because of the Intermediate Value Theorem, which states that if a continuous function has values of opposite signs at the endpoints of an interval, then it must have at least one root within that interval.

Visual Representation:

Key Requirements:

  • The function must be continuous over the interval [a, b]
  • The function values at endpoints must have opposite signs: \(f(a) \cdot f(b) < 0\)

Tip: The bisection method always converges, but it may be slower than other methods like Newton-Raphson. However, its reliability makes it an excellent choice for many applications.

2. The Bisection Algorithm

Step-by-Step Procedure:

  1. Choose an initial interval [a, b] where \(f(a) \cdot f(b) < 0\)
  2. Calculate the midpoint: \(c = \frac{a + b}{2}\)
  3. Calculate \(f(c)\)
  4. Determine the new interval:
    • If \(f(c) = 0\), we've found the root: \(x = c\)
    • If \(f(a) \cdot f(c) < 0\), the root is in [a, c], so set \(b = c\)
    • If \(f(b) \cdot f(c) < 0\), the root is in [c, b], so set \(a = c\)
  5. Repeat steps 2-4 until one of the termination criteria is met

Termination Criteria:

  • Absolute Error: \(|c_{new} - c_{old}| < \epsilon\)
  • Relative Error: \(\frac{|c_{new} - c_{old}|}{|c_{new}|} \times 100\% < \epsilon\)
  • Function Value: \(|f(c)| < \epsilon\)
  • Maximum Iterations: When a predefined number of iterations is reached

Error Analysis:

The maximum error in the bisection method after \(n\) iterations is:

\(E_n = \frac{b - a}{2^{n+1}}\)

To achieve an error less than \(\epsilon\), the number of iterations required is:

\(n \geq \frac{\log(b-a) - \log(\epsilon)}{\log(2)}\)

Common Error: Forgetting to check if the function values at endpoints have opposite signs. Always verify \(f(a) \cdot f(b) < 0\) before starting the algorithm.

3. Example 1: Finding the Root of \(f(x) = x^3 - 2x - 5\)

Problem
Setup
Solution

Find a root of the function \(f(x) = x^3 - 2x - 5\) using the bisection method with a tolerance of \(\epsilon = 0.001\).

This is a common example for the bisection method because the function is continuous and has both positive and negative values.

Step 1: Find an Initial Interval

Let's evaluate the function at several points to find where it changes sign:

x f(x) Sign
1 1 - 2 - 5 = -6 Negative
2 8 - 4 - 5 = -1 Negative
3 27 - 6 - 5 = 16 Positive

Since \(f(2) < 0\) and \(f(3) > 0\), the function changes sign in the interval [2, 3].

The Bisection Setup

For this example:

  • Function: \(f(x) = x^3 - 2x - 5\)
  • Initial interval: [2, 3]
  • Tolerance: \(\epsilon = 0.001\)
  • Maximum iterations: 20 (to be safe)

Let's verify our initial interval meets the requirements:

  • \(f(2) = 2^3 - 2(2) - 5 = 8 - 4 - 5 = -1 < 0\)
  • \(f(3) = 3^3 - 2(3) - 5 = 27 - 6 - 5 = 16 > 0\)
  • Since \(f(2) \cdot f(3) = -1 \cdot 16 = -16 < 0\), the interval contains a root.

Graph of \(f(x) = x^3 - 2x - 5\) showing the root between 2 and 3:

Bisection Method Solution

Iteration 1
\(a = 2\), \(f(a) = -1\)
\(b = 3\), \(f(b) = 16\)
\(c = \frac{a + b}{2} = \frac{2 + 3}{2} = 2.5\)
\(f(c) = f(2.5) = (2.5)^3 - 2(2.5) - 5 = 15.625 - 5 - 5 = 5.625 > 0\)

Since \(f(c) > 0\) and \(f(a) < 0\), the root is in [a, c] = [2, 2.5].

Iteration 2
\(a = 2\), \(f(a) = -1\)
\(b = 2.5\), \(f(b) = 5.625\)
\(c = \frac{a + b}{2} = \frac{2 + 2.5}{2} = 2.25\)
\(f(c) = f(2.25) = (2.25)^3 - 2(2.25) - 5 = 11.39 - 4.5 - 5 = 1.89 > 0\)

Since \(f(c) > 0\) and \(f(a) < 0\), the root is in [a, c] = [2, 2.25].

Iteration 3
\(a = 2\), \(f(a) = -1\)
\(b = 2.25\), \(f(b) = 1.89\)
\(c = \frac{a + b}{2} = \frac{2 + 2.25}{2} = 2.125\)
\(f(c) = f(2.125) = (2.125)^3 - 2(2.125) - 5 = 9.58 - 4.25 - 5 = 0.33 > 0\)

Since \(f(c) > 0\) and \(f(a) < 0\), the root is in [a, c] = [2, 2.125].

Iteration 4
\(a = 2\), \(f(a) = -1\)
\(b = 2.125\), \(f(b) = 0.33\)
\(c = \frac{a + b}{2} = \frac{2 + 2.125}{2} = 2.0625\)
\(f(c) = f(2.0625) = (2.0625)^3 - 2(2.0625) - 5 = 8.77 - 4.125 - 5 = -0.36 < 0\)

Since \(f(c) < 0\) and \(f(b) > 0\), the root is in [c, b] = [2.0625, 2.125].

Iteration 5
\(a = 2.0625\), \(f(a) = -0.36\)
\(b = 2.125\), \(f(b) = 0.33\)
\(c = \frac{a + b}{2} = \frac{2.0625 + 2.125}{2} = 2.09375\)
\(f(c) = f(2.09375) = (2.09375)^3 - 2(2.09375) - 5 = 9.17 - 4.1875 - 5 = -0.02 < 0\)

Since \(f(c) < 0\) and \(f(b) > 0\), the root is in [c, b] = [2.09375, 2.125].

Iteration 6
\(a = 2.09375\), \(f(a) = -0.02\)
\(b = 2.125\), \(f(b) = 0.33\)
\(c = \frac{a + b}{2} = \frac{2.09375 + 2.125}{2} = 2.10938\)
\(f(c) = f(2.10938) = (2.10938)^3 - 2(2.10938) - 5 = 9.38 - 4.21876 - 5 = 0.16 > 0\)

Since \(f(c) > 0\) and \(f(a) < 0\), the root is in [a, c] = [2.09375, 2.10938].

Current interval width: 2.10938 - 2.09375 = 0.01563 > 0.001

Iteration 7
\(a = 2.09375\), \(f(a) = -0.02\)
\(b = 2.10938\), \(f(b) = 0.16\)
\(c = \frac{a + b}{2} = \frac{2.09375 + 2.10938}{2} = 2.10156\)
\(f(c) = f(2.10156) = (2.10156)^3 - 2(2.10156) - 5 = 9.28 - 4.20312 - 5 = 0.07 > 0\)

Since \(f(c) > 0\) and \(f(a) < 0\), the root is in [a, c] = [2.09375, 2.10156].

Current interval width: 2.10156 - 2.09375 = 0.00781 > 0.001

Iteration 8
\(a = 2.09375\), \(f(a) = -0.02\)
\(b = 2.10156\), \(f(b) = 0.07\)
\(c = \frac{a + b}{2} = \frac{2.09375 + 2.10156}{2} = 2.09766\)
\(f(c) = f(2.09766) = (2.09766)^3 - 2(2.09766) - 5 = 9.22 - 4.19532 - 5 = 0.02 > 0\)

Since \(f(c) > 0\) and \(f(a) < 0\), the root is in [a, c] = [2.09375, 2.09766].

Current interval width: 2.09766 - 2.09375 = 0.00391 > 0.001

Iteration 9
\(a = 2.09375\), \(f(a) = -0.02\)
\(b = 2.09766\), \(f(b) = 0.02\)
\(c = \frac{a + b}{2} = \frac{2.09375 + 2.09766}{2} = 2.0957\)
\(f(c) = f(2.0957) = (2.0957)^3 - 2(2.0957) - 5 = 9.20 - 4.1914 - 5 = 0.01 > 0\)

Since \(f(c) > 0\) and \(f(a) < 0\), the root is in [a, c] = [2.09375, 2.0957].

Current interval width: 2.0957 - 2.09375 = 0.00195 > 0.001

Iteration 10
\(a = 2.09375\), \(f(a) = -0.02\)
\(b = 2.0957\), \(f(b) = 0.01\)
\(c = \frac{a + b}{2} = \frac{2.09375 + 2.0957}{2} = 2.09473\)
\(f(c) = f(2.09473) = (2.09473)^3 - 2(2.09473) - 5 = 9.19 - 4.18946 - 5 = 0.00054 > 0\)

Since \(f(c) > 0\) and \(f(a) < 0\), the root is in [a, c] = [2.09375, 2.09473].

Current interval width: 2.09473 - 2.09375 = 0.00098 < 0.001

Since the interval width is less than our tolerance, we stop here.

The root is approximately \(x \approx 2.09473\) with \(f(2.09473) \approx 0.00054\)

We required 10 iterations to achieve the desired tolerance.

4. Example 2: Finding the Root of \(f(x) = x^2 - 2\)

Problem
Setup
Solution

Find the positive root of the function \(f(x) = x^2 - 2\) using the bisection method with a tolerance of \(\epsilon = 0.0001\).

This problem is asking us to find \(\sqrt{2}\) numerically. The exact value is an irrational number, which makes it a good candidate for the bisection method.

Step 1: Find an Initial Interval

Let's evaluate the function at several points to find where it changes sign:

x f(x) Sign
1 1 - 2 = -1 Negative
1.5 (1.5)² - 2 = 2.25 - 2 = 0.25 Positive

Since \(f(1) < 0\) and \(f(1.5) > 0\), the function changes sign in the interval [1, 1.5].

The Bisection Setup

For this example:

  • Function: \(f(x) = x^2 - 2\)
  • Initial interval: [1, 1.5]
  • Tolerance: \(\epsilon = 0.0001\)
  • Maximum iterations: 20 (to be safe)

Let's verify our initial interval meets the requirements:

  • \(f(1) = 1^2 - 2 = -1 < 0\)
  • \(f(1.5) = 1.5^2 - 2 = 2.25 - 2 = 0.25 > 0\)
  • Since \(f(1) \cdot f(1.5) = -1 \cdot 0.25 = -0.25 < 0\), the interval contains a root.

This problem is mathematically equivalent to finding \(\sqrt{2}\), since the root of \(x^2 - 2 = 0\) is \(x = \sqrt{2} \approx 1.4142\).

Graph of \(f(x) = x^2 - 2\) showing the root at \(\sqrt{2}\):

Bisection Method Solution

Iteration 1
\(a = 1\), \(f(a) = -1\)
\(b = 1.5\), \(f(b) = 0.25\)
\(c = \frac{a + b}{2} = \frac{1 + 1.5}{2} = 1.25\)
\(f(c) = f(1.25) = (1.25)^2 - 2 = 1.5625 - 2 = -0.4375 < 0\)

Since \(f(c) < 0\) and \(f(a) < 0\), the root is in [c, b] = [1.25, 1.5].

Iteration 2
\(a = 1.25\), \(f(a) = -0.4375\)
\(b = 1.5\), \(f(b) = 0.25\)
\(c = \frac{a + b}{2} = \frac{1.25 + 1.5}{2} = 1.375\)
\(f(c) = f(1.375) = (1.375)^2 - 2 = 1.890625 - 2 = -0.109375 < 0\)

Since \(f(c) < 0\) and \(f(b) > 0\), the root is in [c, b] = [1.375, 1.5].

Iteration 3
\(a = 1.375\), \(f(a) = -0.109375\)
\(b = 1.5\), \(f(b) = 0.25\)
\(c = \frac{a + b}{2} = \frac{1.375 + 1.5}{2} = 1.4375\)
\(f(c) = f(1.4375) = (1.4375)^2 - 2 = 2.0664 - 2 = 0.0664 > 0\)

Since \(f(c) > 0\) and \(f(a) < 0\), the root is in [a, c] = [1.375, 1.4375].

Iteration 4
\(a = 1.375\), \(f(a) = -0.109375\)
\(b = 1.4375\), \(f(b) = 0.0664\)
\(c = \frac{a + b}{2} = \frac{1.375 + 1.4375}{2} = 1.40625\)
\(f(c) = f(1.40625) = (1.40625)^2 - 2 = 1.9775 - 2 = -0.0225 < 0\)

Since \(f(c) < 0\) and \(f(b) > 0\), the root is in [c, b] = [1.40625, 1.4375].

Iteration 5
\(a = 1.40625\), \(f(a) = -0.0225\)
\(b = 1.4375\), \(f(b) = 0.0664\)
\(c = \frac{a + b}{2} = \frac{1.40625 + 1.4375}{2} = 1.421875\)
\(f(c) = f(1.421875) = (1.421875)^2 - 2 = 2.0217 - 2 = 0.0217 > 0\)

Since \(f(c) > 0\) and \(f(a) < 0\), the root is in [a, c] = [1.40625, 1.421875].

Iteration 12
\(a = 1.4142\), \(f(a) = -0.0000236\)
\(b = 1.41425\), \(f(b) = 0.0000729\)
\(c = \frac{a + b}{2} = \frac{1.4142 + 1.41425}{2} = 1.414225\)
\(f(c) = f(1.414225) = (1.414225)^2 - 2 = 2.0000122 - 2 = 0.0000122 > 0\)

Since \(f(c) > 0\) and \(f(a) < 0\), the root is in [a, c] = [1.4142, 1.414225].

Current interval width: 1.414225 - 1.4142 = 0.000025 < 0.0001

Since the interval width is less than our tolerance, we stop here.

The root is approximately \(x \approx 1.4142\) with \(f(1.4142) \approx -0.0000236\)

We required 12 iterations to achieve the desired tolerance.

The actual value of \(\sqrt{2} \approx 1.41421356237\), so our approximation is quite accurate.
